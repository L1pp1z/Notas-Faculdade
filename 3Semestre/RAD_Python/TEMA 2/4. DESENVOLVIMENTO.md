A utilização de ferramentas de desenvolvimento rápido é essencial para a eficácia da metodologia RAD, facilitando a interação entre desenvolvedores e usuários por meio de protótipos funcionais.

- Usuários podem usar o software e fazer comentários e sugestões
- Com a validação do protótipo os devs montam um plano de trabalho para atender aos requisitos do projeto
- A escolha da linguagem é um fator que aumenta a produtividade

## PYTHON
Python é uma linguagem adequada para a aplicação da metodologia RAD.
- <span style="color:rgb(255, 192, 0)">PORTABILIDADE:</span> Pode-se trabalhar em diversos SO
- <span style="color:rgb(255, 192, 0)">LICENÇA DE CÓDIGO ABERTO:</span> Distribuições são gratuitas e de código aberto sem restrições de licença
- <span style="color:rgb(255, 192, 0)">INTEGRAÇÃO:</span> Pacotes que abrangem banco de dados, web, interfaces gráficas, entre outros.
- <span style="color:rgb(255, 192, 0)">DESENVOLVIMENTO RÁPIDO:</span> Sintaxe mais simples e vasta opção de pacotes e frameworks que auxiliam na velocidade
- <span style="color:rgb(255, 192, 0)">LINGUAGUEM INTERPRETADA:</span> Interatividade mais simples
- <span style="color:rgb(255, 192, 0)">COMUNIDADE:</span> Ampla e participativa

## Ferramentas
A utilização de ferramentas (frameworks) Python é indispensável na metodologia RAD.
Frameworks como Django, Flask e PyQt, facilitam a criação de protótipos

### Tipos de Frameworks
- <span style="color:rgb(255, 192, 0)">FULL-STACK:</span> Solução completa de todos os requisitos para o desenvolvedor
- <span style="color:rgb(255, 192, 0)">MICROFRAMEWORK:</span> Oferece uma quantidade mínima de serviços. Utilizado para recebimento de solicitações, roteamento, despacho e retorno de requisições
- <span style="color:rgb(255, 192, 0)">FRAMEWORK ASSÍNCRONOS:</span> Lida com grande conjunto de conexões simultâneas

#### Frameworks GUI
Conjunto de pacotes que auxiliam na criação de interfaces gráficas. Exemplos:
- <span style="color:rgb(255, 192, 0)">Tkinter:</span> permite desenvolver interfaces gráficas
- <span style="color:rgb(255, 192, 0)">PyQt:</span> biblioteca de componentes gráficos do Python
- <span style="color:rgb(255, 192, 0)">PySide:</span> software livre que executa nos sistemas operacionais Windows, Mac e Linux
- <span style="color:rgb(255, 192, 0)">Kivy:</span> permite desenvolver aplicativos “multi-touch” para dispositivos móveis e computadores
- <span style="color:rgb(255, 192, 0)">wxPython:</span> kit de ferramentas de código aberto que possui suporte para as plataformas Windows de 32 bits, Unix e Mac OS X

###### PRINCIPAIS COMPONENTES

	- <span style="color:rgb(0, 176, 80)">w=Button(master,option=value)</span>

	- <span style="color:rgb(0, 176, 80)">w=Canvas(master, option=value)</span>
- <span style="color:rgb(255, 192, 0)">CHECKBUTTON:</span> É aplicado para selecionar uma opção
	- <span style="color:rgb(0, 176, 80)">w=CheckButton(master, option=value)</span>
- <span style="color:rgb(255, 192, 0)">ENTRY:</span> usado para entrar texto
	- <span style="color:rgb(0, 176, 80)">w=Entry(master, option=value)</span>
- <span style="color:rgb(255, 192, 0)">FRAME:</span> usado para agrupar e organizar componentes
	- <span style="color:rgb(0, 176, 80)">w=Frame(master, option=value)</span>
- <span style="color:rgb(255, 192, 0)">LABEL:</span> exibe uma caixa onde se pode inserir texto
	- <span style="color:rgb(0, 176, 80)">w=Label(Master, option=value)</span>
- <span style="color:rgb(255, 192, 0)">MENUBUTTON:</span> É usado para criar “menus de topo”
	- <span style="color:rgb(0, 176, 80)">w=MenuButton(Master, option=value)</span>
- <span style="color:rgb(255, 192, 0)">MENU:</span> Cria menu
	- <span style="color:rgb(0, 176, 80)">w=Menu(master, option=value)</span>
- <span style="color:rgb(255, 192, 0)">MESSAGE:</span> usado para exibir múltiplas linhas sem texto editável
	- <span style="color:rgb(0, 176, 80)">w=Message(master, option=value)</span>
- <span style="color:rgb(255, 192, 0)">SCALE:</span> componente deslizante que permite selecionar valores de uma escala específica
	-  <span style="color:rgb(0, 176, 80)">w=Scale(master, option=value)</span> 

#### Frameworks WEB:
Conjunto de pacotes que habilitam os desenvolvedores a desenvolver aplicações para web, ou serviços

- <span style="color:rgb(255, 192, 0)">ROTEAMENTO DE URL:</span> Trata de solicitações HTTP recebidas
- <span style="color:rgb(255, 192, 0)">OBJETOS DE SOLICITAÇÃO E RESPOSTA:</span> Agrupa as informações recebidas, ou enviadas, para o navegador de um usuário.
- <span style="color:rgb(255, 192, 0)">TEMPLATE ENGINE:</span> Usados como um formato intermediário para produzir um ou mais formatos de saída, normalmente, HTML, XML ou PDF
- <span style="color:rgb(255, 192, 0)">SERVIDOR WEB DE DESENVOLVIMENTO:</span> Executa um servidor HTTP em máquinas de desenvolvimento.

Exemplos:
- <span style="color:rgb(255, 192, 0)">Django:</span> utilizado criar sites complexos baseados em dados. Possui modelos, bibliotecas e APIs que dão suporte na criação de projetos de desenvolvimento da web escaláveis
- <span style="color:rgb(255, 192, 0)">TurboGear:</span> usa a arquitetura MVC (Model View Control), que ajuda no rápido desenvolvimento de aplicações web
- <span style="color:rgb(255, 192, 0)">Flask:</span> suporta o envio de solicitações REST
- <span style="color:rgb(255, 192, 0)">Bottle:</span> distribuído como um módulo de arquivo único, sem dependências além da biblioteca padrão do Python. Suporta o envio de solicitações com suporte a URL, bancos de dados de chave/valor e modelos e um servidor HTTP interno
- <span style="color:rgb(255, 192, 0)">CherryPy:</span> fornece as funcionalidades CRUD (Criar, Recuperar, Atualizar e Excluir)
- <span style="color:rgb(255, 192, 0)">Falcon:</span> utilizado para o desenvolvimento de programas de pequena escala; usa a arquitetura REST

#### Outros Frameworks
- <span style="color:rgb(255, 192, 0)">Numpy:</span> operações matemáticas e lógicas em matrizes
- <span style="color:rgb(255, 192, 0)">Pandas:</span> aplicada, especialmente, na análise de dados
- <span style="color:rgb(255, 192, 0)">Matplotlib:</span> biblioteca de plotagem que auxilia na criação de gráficos e plotagens 2D, que incluem gráficos de barras, histogramas e muitos outros
